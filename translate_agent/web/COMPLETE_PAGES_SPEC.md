```
   │ ••••••••          [👁️]     │
   │                              │
   │ 密码强度: [████░░] 中等      │

4. Confirm Password (确认密码)
   ┌─────────────────────────────┐
   │ 🔒 确认密码                 │

   │ ••••••••                    │

   ☑ 我同意 [服务条款] 和 [隐私政策]

   ┌─────────────────────────────┐

   └─────────────────────────────┘

Footer:
   已有账号？ [登录]
```

### 交互细节

#### 密码强度指示器

**实时验证** (输入时):
```
弱:   [█   ]  1-3 字符
中:   [███ ]  4-7 字符
强:   [████]  8+ 字符 + 特殊字符
```

**颜色变化**:
- 弱: error (红色)
- 中: warning (橙色)
- 强: success (绿色)

**动画**:
- 平滑过渡 (300ms)
- 条纹填充动画

#### 确认密码验证

**输入时实时比对**:
- 匹配: ✅ 绿色勾选
- 不匹配: ❌ 红色错误 + "密码不一致"

#### 服务条款 Checkbox

**默认**: 未选中
**点击**:
- 显示勾选动画
- 按钮从 disabled 变为 enabled

#### 注册 Button

**状态**:
1. **默认**:
   - 渐变背景
   - disabled (未勾选条款)

2. **表单有效**:
   - enabled
   - 可点击

3. **Loading**:
   - [⟳] 注册中...
   - disabled

4. **成功**:
   - [✓] 注册成功
   - 延迟 1秒
   - 跳转登录页

### 表单验证规则

```typescript
{
  name: {
    required: "姓名不能为空",
    minLength: { value: 2, message: "至少2个字符" }
  },
  email: {
    required: "邮箱不能为空",
    pattern: /^[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,}$/i
  },
  password: {
    required: "密码不能为空",
    minLength: { value: 8, message: "至少8个字符" },
    pattern: /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)/
  },
  confirmPassword: {
    required: "请确认密码",
    validate: (value) => value === password
  },
  terms: {
    required: "必须同意服务条款"
  }
}
```

### 分步注册（可选优化）

**Step 1: 基本信息**
- 姓名
- 邮箱

**Step 2: 设置密码**
- 密码
- 确认密码

**Step 3: 完善资料**（可选）
- 头像上传
- 公司/组织
- 职业

**优点**:
- 减少表单压力
- 提高完成率
- 更好的用户体验
